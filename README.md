# romance
A website with flowers that never stop blooming.

Setup:
Option 1: open index.html with other files in your browser
Option 2: open link that converted these files to one website: luvu3.netlify.app

No additions needed. Basic html, css and js code just with more details e.g. each petal has its own lines.
